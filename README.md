# Portfolio

Portfolio personnel en **React.js + Vite + Tailwind CSS**, thème noir et blanc.

## Démarrage

```bash
npm install
npm run dev
```

Ouvre [http://localhost:5173](http://localhost:5173) dans le navigateur.

## Personnalisation

- **Hero** : `src/components/Hero.jsx` — change ton prénom et le sous-titre.
- **Expérience** : `src/components/Experience.jsx` — modifie le tableau `experiences` (titre, entreprise, période, description).
- **Projets** : `src/components/Projects.jsx` — modifie le tableau `projects` (titre, description, `url`). Chaque carte est un lien cliquable vers le projet.
- **Footer** : `src/components/Footer.jsx` — email et lien GitHub.

## Build

```bash
npm run build
```

Les fichiers de production sont dans `dist/`.
